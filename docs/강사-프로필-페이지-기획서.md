# 디에듀 101 목록 페이지 기획서

## 1. 개요

### 목적
신규 가입한 강사들의 프로필과 스터디룸을 공개하여, 학생 및 학부모가 강사를 탐색하고 선택할 수 있는 공간을 제공합니다.

### 대상 사용자
- **주 사용자**: 학생, 학부모 (강사 탐색 및 선택)
- **부 사용자**: 신규 가입 강사 (자신의 프로필 노출 인지)

### 주요 기능
1. 신규 강사 프로필 목록 조회
2. 공개 스터디룸 목록 조회
3. 강사/스터디룸 상세 페이지로 이동
4. 정렬 및 필터링 기능

## 3. API 명세

### 3.1 공개 강사 목록 조회

**엔드포인트**: `GET /public/teachers`

**요청 파라미터**:
```typescript
{
  page?: number;           // 페이지 번호 (기본값: 0)
  size?: number;           // 페이지 크기 (기본값: 20)
  sort?: 'NEWEST' | 'POPULAR' | 'RATING';  // 정렬 기준
  isNewTeacher?: boolean; // 신규 강사만 조회 (30일 이내 가입)
}
```

**응답 구조**:
```typescript
{
  content: [
    {
      id: number;                    // 강사 ID
      name: string;                  // 강사 이름
      nickname?: string;            // 닉네임
      profileImageUrl?: string;     // 프로필 이미지 URL
      bio?: string;                 // 자기소개
      specialties?: string[];       // 전문 분야 (예: ['수능 국어', '고등학교'])
      experience?: string;          // 경력 (예: '10년')
      rating?: number;             // 평점 (0-5)
      reviewCount?: number;         // 리뷰 개수
      studyRoomCount: number;       // 보유 스터디룸 개수
      joinedAt: string;            // 가입일 (ISO 8601)
      isNewTeacher: boolean;       // 신규 강사 여부 (30일 이내)
    }
  ],
  totalElements: number;    // 전체 개수
  totalPages: number;       // 전체 페이지 수
  pageNumber: number;      // 현재 페이지
  size: number;           // 페이지 크기
}
```

**정렬 기준**:
- `NEWEST`: 가입일 기준 최신순
- `POPULAR`: 스터디룸 개수 기준 인기순
- `RATING`: 평점 기준 높은순

**필터링**:
- `isNewTeacher=true`: 가입일이 30일 이내인 강사만 조회

---

### 3.2 공개 스터디룸 목록 조회

**엔드포인트**: `GET /public/study-rooms`

**요청 파라미터**:
```typescript
{
  page?: number;           // 페이지 번호 (기본값: 0)
  size?: number;           // 페이지 크기 (기본값: 20)
  sort?: 'NEWEST' | 'POPULAR';  // 정렬 기준
  teacherId?: number;     // 특정 강사의 스터디룸만 조회
}
```

**응답 구조**:
```typescript
{
  content: [
    {
      id: number;                    // 스터디룸 ID
      name: string;                  // 스터디룸 이름
      description: string;           // 설명
      teacherId: number;             // 강사 ID
      teacherName: string;           // 강사 이름
      teacherProfileImageUrl?: string; // 강사 프로필 이미지 URL
      studyRoomImageUrl?: string;    // 스터디룸 대표 이미지 URL
      subjectType?: string;          // 과목 (예: '국어', '수학')
      grade?: string;                // 대상 학년 (예: '고3', '중3')
      studentCount?: number;         // 학생 수
      visibility: 'PUBLIC' | 'PRIVATE'; // 공개 여부
      createdAt: string;             // 생성일 (ISO 8601)
    }
  ],
  totalElements: number;
  totalPages: number;
  pageNumber: number;
  size: number;
}
```

**정렬 기준**:
- `NEWEST`: 생성일 기준 최신순
- `POPULAR`: 학생 수 기준 인기순

**필터링**:
- `teacherId`: 특정 강사의 스터디룸만 조회

---

### 3.3 강사 공개 프로필 상세 조회

**엔드포인트**: `GET /public/teachers/{teacherId}`

**응답 구조**: 위의 강사 목록 항목과 동일 (단일 객체)

## 4. 데이터 요구사항

### 4.1 강사 프로필 데이터
- **필수**: id, name, studyRoomCount, joinedAt
- **선택**: nickname, profileImageUrl, bio, specialties, experience, rating, reviewCount
- **계산 필드**: isNewTeacher (joinedAt 기준 30일 이내 여부)

### 4.2 스터디룸 데이터
- **필수**: id, name, description, teacherId, teacherName, visibility, createdAt
- **선택**: studyRoomImageUrl, teacherProfileImageUrl, subjectType, grade, studentCount
- **필터링**: visibility가 'PUBLIC'인 스터디룸만 조회

### 4.3 권한
- 모든 엔드포인트는 **인증 불필요** (공개 API)
- 로그인하지 않은 사용자도 접근 가능

## 5. UI/UX 요구사항

### 5.1 강사 카드
- 프로필 이미지 (없으면 이니셜 표시)
- 이름 + "선생님"
- 전문 분야 태그 (최대 3개)
- 자기소개 (2줄 말줄임)
- 통계: 스터디룸 개수, 평점/리뷰 수
- 신규 강사 배지 (NEW)

### 5.2 스터디룸 카드
- 스터디룸 대표 이미지 (없으면 기본 이미지/이모지)
- 강사 프로필 이미지 + 이름
- 스터디룸 이름
- 설명 (2줄 말줄임)
- 통계: 학생 수, 학년

### 5.3 정렬 옵션
- 강사 탭: 최신순, 인기순, 평점순
- 스터디룸 탭: 최신순, 인기순

### 5.4 반응형 디자인
- 모바일: 1열
- 태블릿: 2-3열
- 데스크톱: 3-4열
